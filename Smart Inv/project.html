⬡ Stack
Requirements
Frontend: Vanilla HTML5, CSS3, JavaScript
Storage: Browser LocalStorage
Backend: Optional Firebase or REST API integration
Responsive UI: CSS Grid, mobile-first
Fonts: Google Fonts (Syne + DM Sans)
AI: Client-side analysis engine
⬡ Design Principles
Constraints
Performance: Instant updates, no page reloads
Simplicity: Single-page, no build tools needed
Scalability: Handles hundreds of products via filtered rendering
Data safety: All writes to LocalStorage immediately
Accessibility: Semantic HTML, keyboard navigation
Privacy: No external data transmission
